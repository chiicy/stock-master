# 板块面分析框架

## 分析维度

### 1. 板块定位
- 所属行业
- 所属概念
- 是龙头、次龙头、跟风还是边缘票

### 2. 板块强弱
- 板块涨跌幅
- 板块资金流方向
- 龙头股表现
- 板块成交额与上涨家数
- 板块内资金是否向龙头集中

### 3. 相对位置
- 个股在板块中的相对强弱
- 是否跑赢板块
- 是否出现板块强、个股弱或板块弱、个股强

### 4. 轮动与持续性
- 板块是否处于热点轮动中
- 是趋势性主线还是短期脉冲
- 近 3~5 日强度是否持续
- 日内切换速度快不快，是否属于高频轮动而非主线

## 输出模板

### 板块面结论
- 板块热度：高 / 中 / 低
- 个股地位：龙头 / 跟风 / 边缘
- 板块趋势：上升 / 下降 / 震荡
- 次日观察点：看龙头强度、看资金是否回流、看板块是否扩散
- 风险提示：板块退潮时个股弹性会快速收缩
- 若只有单次板块资金流，不能写出“近 5 日轮动持续性”结论
