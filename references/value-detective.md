# 深度价值分析主框架

> 仅在用户明确要求“财报侦探”“价值投资”“深度估值”“ROIC/WACC/DCF”且数据足够时启用。本文件提供深度价值分析的报告骨架与 1→22 清单。

## 启用条件

至少尽量补齐以下材料；缺什么就明确写“数据缺口”并降级：
- 股票代码、公司全称、交易所 / 市场、币种与单位
- 最近 3 年利润表、资产负债表、现金流量表
- 财报摘要 / MD&A / 管理层讨论
- 当前股价 / 市值 / PE 与最近 30 天股价区间
- 行业分类与至少 3 家可比公司

## 证据链写法

深度模式下，每个关键结论都尽量按这个顺序写：
- 数据来源：文件 / 表名 / 页码 / 口径
- 计算：公式 + 代入值
- 结论：因为……所以……
- 反证 / 替代解释：还有什么别的解释，如何验证

不要只写“表现不错”“存在风险”这种空话。

## 报告骨架

### 1. 基础信息
- 公司、代码、行业、币种、单位、会计准则、覆盖财年
- 当前股价 / 市值 / PE 的日期
- 最近 30 天股价区间
- 数据来源与口径说明

### 2. 数据总表
- 三年利润表关键项
- 三年资产负债表关键项
- 三年现金流关键项
- 三年关键比率
- 同业横向对比

### 3. 一页结论
- 生意本质一句话
- 财务透视：增长 / 盈利 / 现金
- 风险与竞争力
- 管理层兑现度
- 综合结论：好生意 / 一般生意 / 差生意；好价格 / 一般价格 / 坏价格

## 1→22 侦探式清单

### 1. 生意本质
- 公司靠什么赚钱，客户是谁，产业链位置在哪
- 关键资产是什么：品牌、专利、牌照、渠道、规模、数据
- 用一句大白话类比说清业务模型

### 2. 深入财务数据分析
- 利润表：营收增长、毛利率、费用率、增长质量
- 资产负债表：现金、债务、应收、存货、商誉、营运资本
- 现金流量表：CFO、CAPEX、FCF、CFO / 净利润
- 关键比率：ROE 杜邦、应收周转、存货周转、ROIC

### 3. 风险与竞争力
- 只挑 1 个最大“颠覆性风险”
- 再列 1~3 个主要风险，写触发条件和监控指标
- 护城河是否真的转化成定价权或持续降本

### 4. 管理层洞察
- 管理层最近几年主叙事是什么
- 过去承诺是否兑现
- 更像“兑现型管理层”还是“故事型管理层”

### 5. 综合与对比
- 与至少 3 家同业比毛利率、净利率、ROE、ROIC、负债和现金流
- 看公司自己 3 年甚至 5~10 年趋势
- 判断行业处在导入 / 成长 / 成熟 / 衰退哪个阶段

### 6. 资本配置能力
- 并购、扩产、分红、回购、剥离这些动作有没有创造价值
- 资本配置风格是克制、激进还是摇摆

### 7. 新赛道与第二曲线
- 新业务能不能成为真实增量
- 与主业是否协同，天花板在哪里

### 8. 市场预期 vs 你的判断
- 当前价格大致反映了什么预期
- 你的核心分歧点是什么

### 9. 市盈率真伪与隐含增长
- 当前 PE 倒推出的增长是否现实
- 是泡沫、误杀，还是低估陷阱

### 10. 宏观环境传导
- 利率、通胀、汇率、政策这些变量怎么传到三张表
- 尽量粗量化，而不是只写口号

### 11. 护城河压力测试
- 如果打价格战或有强新进入者，公司能不能守住利润和份额
- 最好找一个历史案例做参照

### 12. 近 30 天股价驱动
- 列出最近 1~3 个主要驱动
- 区分短期盈利、长期价值和情绪波动

### 13. 评分
- 各项打分，再给总分
- 解释权重，不要只报总分

### 14. 买入性价比
- 安全边际、确定性、下行风险三维判断

### 15. 情景化介入计划
- 基准 / 乐观 / 悲观三种情景
- 触发条件 + 仓位 / 加减仓框架

### 16. 客户 / 供应商集中度
- 前五大客户 / 供应商占比
- 是否存在议价权弱、单一依赖或景气先行风险

### 17. ROIC vs WACC
- 判断公司是在持续创造价值还是消耗资本
- 口径要写清

### 18. 股权激励 / 重要股东交易 / 审计意见
- 行权条件有没有诱导短期利润管理
- 大股东增减持和估值 / 业绩节点是否一致
- 审计意见是否有警报

### 19. 业务模式是否足够好
- 简单、清晰、轻资产、强现金流、定价权、护城河这几项分别打证据

### 20. 分红与股息率可持续性
- 分红是否被 FCF 覆盖
- 是否靠举债分红

### 21. 利润敏感性
- 利润最受哪些变量驱动：销量、ASP、原材料、汇率、一次性损益
- 尽量量化敏感度

### 22. 情绪位置检查
- 现在是无人问津、冷静定价，还是人声鼎沸
- 最终一句话：好公司 / 好价格 是否同时满足

## 输出要求

- 深度模式不是把 22 条都写成长篇大论，而是按结构逐项过一遍
- 数据不足时直接写：
  - 缺了什么
  - 会影响哪部分判断
  - 下一步该补什么
- 结尾保留一句合规提示：仅供研究学习，不构成个性化投资建议
