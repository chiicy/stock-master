# 资金面分析框架

## 分析维度

### 1. 主力资金
- 主力净流入 / 净流出
- 超大单、大单、中单、小单结构
- 连续流入 / 连续流出天数

### 2. 北向资金
- 北向净买入 / 净卖出
- 近 N 日趋势
- 作为市场风险偏好辅助指标

### 3. 板块资金
- 所属板块资金流方向
- 个股是否和板块同步
- 资金是否在板块内部向龙头集中

### 4. 交易性资金
- 龙虎榜、融资融券、异常放量
- 高波动票要注意游资主导特征

## 输出模板

### 资金面结论
- 主力态度：流入 / 流出 / 观望
- 资金强度：强 / 中 / 弱
- 板块配合：有 / 无
- 风险提示：缩量上行、放量下跌、冲高回落等信号

## 写作要求
- 资金面偏短中周期，不宜单独定义长期价值
- 个股强、板块弱时，说明个股更像事件驱动
- 板块强、个股弱时，说明资金未选中它
