# 技术面分析框架

## 使用原则

- 默认面向 A 股单股技术分析；如果用户给的是其他市场，先说明边界。
- 技术面输出必须区分：
  - 事实层：价格、均线、RSI、MACD、KDJ、布林带、量价、关键位
  - 推断层：偏多 / 偏空 / 震荡、未来 3 个交易日基准判断、操作建议
- 只使用真实返回的指标。若当前数据链路没有 ADX、EMA50 / EMA200、缺口明细等字段，不要编造数值。
- 用户明确要求“像虾评一样”“支撑压力”“有没有缺口”“未来三天”，就按下面的深度技术面结构输出。

## 分析维度

### 1. 趋势与结构
- MA5 / MA10 / MA20 / MA60
- 均线多头 / 空头 / 粘合
- 当前价格相对各均线位置
- 若工具提供 EMA50 / EMA200：
  - 价格 > EMA50 > EMA200：中期偏多
  - 价格 < EMA50 < EMA200：中期偏空
- 若用户还关心市场环境，补看指数、板块和北向资金，避免脱离大盘孤立解读

### 2. 动量与摆动指标
- RSI：>70 偏超买，<30 偏超卖
- RSI < 20 或 > 80 视为极端值，要额外提示“波动与回撤风险”
- MACD：金叉 / 死叉 / 柱体扩张 / 柱体收敛
- MACD 在零轴上方更偏趋势延续，在零轴下方更偏反弹 / 走弱框架
- KDJ：J 值过热，K/D 交叉
- 若工具提供 ADX：
  - ADX > 25：趋势明确，优先顺势
  - ADX < 20：更像震荡，信号可靠性下降
- 若工具提供 EMA50 / EMA200：
  - 价格 > EMA50 > EMA200：中期偏多
  - 价格 < EMA50 < EMA200：中期偏空

### 3. 波动与区间
- 布林带：上轨 / 中轨 / 下轨
- 带宽扩张代表波动放大
- 带宽明显收窄代表临近变盘，尤其适合和量能一起看
- 近 20 / 60 日高低点
- 箱体上沿 / 下沿是否被有效突破，要看收盘位置和量能确认

### 4. 量价关系
- 放量上涨 / 缩量上涨 / 放量下跌 / 缩量下跌
- 量比 / 近 5 日成交量 vs 近 20 日成交量
- 量价背离
- 冲高回落放量通常是短线分歧加大，缩量上攻则需警惕跟风不足

### 5. 关键位与缺口
- 支撑位：前低、均线、箱体下沿、布林下轨
- 压力位：前高、均线、箱体上沿、布林中轨 / 上轨
- 破位 / 突破是否有效，需结合量能和收盘确认
- 缺口分析规则：
  - 若工具没有直接给缺口字段，不要编造“向上缺口 xx 元”之类细节
  - 可以保守写为“当前工具未直接返回缺口明细，因此只做保守观察”
  - 若近期价格结构明显跳空，可只描述“未回补缺口通常会成为短线支撑 / 压力观察位”

### 6. 未来 3 个交易日判断
- 只做方向性判断：上涨倾向 / 震荡倾向 / 回落倾向
- 依据优先顺序：
  - 均线结构
  - MACD 与 RSI
  - 布林带位置与带宽
  - 量价是否配合
  - 支撑 / 压力是否临近
- 必须给失效条件：
  - 跌破关键支撑
  - 放量走弱
  - 指标重新死叉
  - 无法有效站回关键均线 / 压力位

## 输出模板

### 技术面结论
- 事实层
  - 趋势：上涨 / 下跌 / 震荡
  - 强度：强 / 中 / 弱
  - 均线结构：多头 / 空头 / 粘合
  - 关键位：支撑 X，压力 Y
  - 指标信号：RSI、MACD、KDJ、布林带的核心信号
  - 缺口观察：若数据不足，要明确写缺失
- 推断层
  - 未来 3 个交易日：上涨 / 震荡 / 回落倾向
  - 主要依据：2~4 条
  - 风险提示：什么条件出现后当前判断失效
  - 操作建议：短线 / 已持仓分开写，避免绝对化措辞

## 写作要求
- 至少引用 3 个证据，不要只报一个 RSI 就下结论
- 趋势和位置要分开写：趋势弱不等于立刻反弹
- 对短线判断时，优先看 MA5/MA10、量能、MACD 柱体
- 对中线判断时，优先看 MA20/MA60、平台高低点
- 当前若 quote 快照缺字段，要明确写“已退回到 K 线与可用 quote 字段做判断”
- 明确区分“工具直接返回的事实”和“基于指标的推断”
- 当大盘、板块、个股出现相互矛盾的信号时，要把矛盾点写出来
- 当前工具拿不到缺口明细时，不要硬做精确缺口分析
