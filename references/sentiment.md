# 情绪面分析框架

## 分析维度

### 1. 个股交易情绪
- 换手率高低
- 量比高低
- 当日振幅
- 是否接近涨停 / 跌停

### 2. 市场情绪
- 涨停数 / 跌停数
- 连板高度
- 炸板率
- 赚钱效应
- 恐慌 / 贪婪倾向

### 3. 热度与关注度
- 是否热门题材
- 是否人气股 / 龙头股 / 跟风股
- 是否存在高位情绪退潮风险

## 输出模板

### 情绪面结论
- 市场情绪：恐慌 / 中性 / 贪婪
- 个股热度：高 / 中 / 低
- 情绪趋势：升温 / 降温 / 分化
- 观察点：涨停家数、连板高度、炸板率是否同步改善
- 风险提示：高换手不等于强，缩量板也可能脆弱

## 写作要求
- 情绪面偏交易，不要与基本面混写
- 热度高通常意味着弹性高，也意味着回撤快
- 当前若缺连板高度、炸板率、封单额等字段，只能保守写涨跌停广度，不能伪装成完整情绪周期分析
