# 深度价值公式与口径

> 做深度基本面时，优先引用本文件口径，避免同一份报告里公式前后不一致。

## 通用原则

- 先写清单位、币种、会计准则
- 周转类、ROE、ROIC 尽量用平均值口径
- 公式之外，必须解释口径和假设

## 增长与利润率

- 同比增长：`(本期 - 上期) / 上期`
- 三年 CAGR：`(FY3 / FY1)^(1/2) - 1`
- 毛利率：`(营收 - 成本) / 营收`
- 营业利润率：`EBIT / 营收`
- 净利率：`净利润 / 营收`

## 周转与营运效率

- 平均应收：`(期初应收 + 期末应收) / 2`
- 应收周转率：`营收 / 平均应收`
- DSO：`365 / 应收周转率`

- 平均存货：`(期初存货 + 期末存货) / 2`
- 存货周转率：`营业成本 / 平均存货`
- DIO：`365 / 存货周转率`

- 现金转换周期可选做：
  - `CCC = DSO + DIO - DPO`
  - DPO 若采购额缺失，可用 `365 / (营业成本 / 平均应付)` 近似，但要写明近似

## ROE 与杜邦

- `ROE = 净利润 / 平均股东权益`
- 杜邦三拆：
  - 净利率：`净利润 / 营收`
  - 总资产周转：`营收 / 平均总资产`
  - 权益乘数：`平均总资产 / 平均股东权益`
- `ROE = 净利率 × 总资产周转 × 权益乘数`

写法要点：
- 高 ROE 到底来自高利润率、高周转还是高杠杆
- 如果主要来自高杠杆，要把可持续性和融资风险单列

## ROIC

推荐写法：
- `NOPAT = EBIT × (1 - 税率)`
- 简化投入资本：`股东权益 + 有息负债 - 超额现金`
- `ROIC = NOPAT / 平均投入资本`

如果数据更完整，也可以用经营口径投入资本，但同一份报告必须统一。

## WACC

- `WACC = E/V × Re + D/V × Rd × (1 - T)`
- `V = E + D`
- 股权成本可用粗化 CAPM：
  - `Re = Rf + β × ERP`
- 债务成本可粗估：
  - `Rd ≈ 利息费用 / 平均有息负债`

要求：
- 写明无风险利率、风险溢价、税率和债务成本是假设还是引用
- 若只能区间估值，就给区间，不要硬报精确值

## DCF

FCFF 口径优先：
- `FCFF = EBIT × (1 - T) + 折旧摊销 - CAPEX - ΔNWC`

终值常用：
- `TV = FCF_(n+1) / (WACC - g)`

估值表达要求：
- 至少给三组假设：增长、利润率、再投资
- 给敏感性，而不是只给单点结果
- 如果终值占比过高，要提醒“估值高度依赖终值假设”

## PE 倒推隐含增长

当现金流数据不足、但用户仍然想快速判断估值是否离谱时，可做粗算：
- `P/E ≈ (1 - 再投资率) / (r - g)`
- 反推：`g ≈ r - (1 - 再投资率) / (P/E)`

注意：
- 这是量级判断，不是精确定价
- 一定要拿行业历史增速和公司自身历史验证

## 输出约束

- 公式只是起点，不是结论
- 每个公式后面都要补一句“这说明什么”
- 当数据口径不一致时，先解决口径，再算指标
