# 基本面分析框架

## 使用模式

### 轻量模式

适用于只拿到部分财务指标、财报摘要或当期公开数据时：
- 可以做盈利能力、成长性、现金流质量、资产负债结构、轻量估值判断
- 不要假装完成深度价值分析
- DCF、WACC、完整杜邦、同业 Top3 对比、管理层兑现度等内容，只能点到为止或直接说明数据不足

### 深度价值模式

适用于用户明确要求价值投资 / 深度估值 / 估值报告，且数据足够时：
- 先确认股票代码、公司全称、币种与单位、最近 3 年三张表、财报摘要 / MD&A、当前价格区间、至少 3 家同业
- 先做数据标准化：会计准则、口径、单位、是否追溯重述
- 所有结论尽量写成“因为……（数据/事实/计算）所以……（结论）”
- 每个关键结论都要附带反证或替代解释
- 深度模式下必须再读：
  - `references/value-detective.md`
  - `references/financial-metrics.md`
  - `references/forensics-esg.md`

## 分析维度

### 1. 生意本质与行业位置
- 公司靠什么赚钱，客户是谁，处于产业链哪个位置
- 护城河来自品牌、成本、渠道、技术、牌照还是转换成本
- 是否具备定价权，证据是否能从毛利率、份额、复购或提价后销量中体现

### 2. 盈利能力
- ROE：>15% 优秀，>10% 良好
- 高 ROE 要拆开看：来自高净利率、高周转还是高杠杆
- ROIC：看资本使用效率
- 毛利率 / 净利率：看盈利质量与行业地位

### 3. 成长性
- 营收增长率
- 净利润增长率
- 扣非净利润增长率
- 连续性比单年高增长更重要
- 能拆就拆量 / 价 / 结构 / 并表 / 汇率；不能拆就说明数据缺口

### 4. 三表联动与财务健康
- 资产负债率
- 经营现金流 / 自由现金流
- 应收、存货、商誉是否异常膨胀
- 经营现金流 vs 净利润：利润有没有含金量
- 现金 vs 有息负债：是否净现金 / 净负债，是否存在“存贷双高”
- 营运资本：应收 + 存货 - 应付 的趋势是否挤压现金流

### 5. 估值
- PE / PB / PEG
- 历史分位
- 行业对比
- 数据不足时不要硬做 DCF
- 若只能做粗算，可用“PE 倒推隐含增长”判断市场预期是否现实
- 能做 DCF 时，必须写清假设、WACC、终值与敏感性，而不是给单点答案

### 6. 深度价值方法
- ROE 杜邦拆解：净利率 × 总资产周转 × 权益乘数
- ROIC vs WACC：判断公司是在持续创造价值，还是靠资本堆出来
- 同业横向对比：至少 3 家同业，对比毛利率、净利率、ROE、ROIC、负债和现金流
- 管理层兑现度：战略叙事和财务结果是否一致
- 逆向质疑：至少提出 2 个关键反问，并用数据验证 / 证伪
- 近 30 天股价驱动：把近期上涨 / 下跌拆成盈利、事件、政策、情绪四类
- 情景化介入计划：基准 / 乐观 / 悲观三种情景，而不是一句“可买 / 不可买”

### 7. 风险点
- 商誉减值风险
- 存货跌价风险
- 关联交易风险
- 现金流与利润背离
- 审计意见、股权激励、大股东增减持、客户 / 供应商集中度
- ESG 中最可能影响长期价值的 1~2 个因子

## 输出模板

### 基本面结论
- 数据完整性：完整 / 部分完整 / 明显不足
- 分析模式：轻量模式 / 深度价值模式
- 生意本质：一句话说清公司怎么赚钱
- 财务透视
  - 增长：营收 / 利润是否可持续
  - 盈利：毛利率 / ROE / ROIC / 费用率代表什么
  - 现金：CFO / 净利润 / FCF 说明什么
- 估值判断：高估 / 合理 / 低估 / 数据不足
- 同业对比：领先 / 持平 / 落后，依据是什么
- 优势：2~4 条
- 风险：2~4 条
- 适用周期：短线不作为主判断，偏中长线

## 写作要求
- 先说数据是否完整
- 缺财报时，只能做轻量判断，不要装作完成深度价值分析
- 不要把高 ROE 自动等同于好公司，要结合杠杆和现金流
- 关键结论尽量落到具体数字、比率、公式或已知事实
- 有 DCF / WACC / ROIC 时必须写清口径和假设
- 至少给 1~2 个逆向质疑点，而不是只顺着公司故事往下写
- 深度模式要尽量覆盖“生意本质 -> 三表联动 -> 估值 -> 风险排雷 -> 介入计划”这条完整链路
- 结尾保留一句简短风险提示：仅供研究学习，不构成个性化投资建议
- 若当前只能满足深度模式门槛的一部分，但拿不到 MD&A / 同业 / 估值关键输入，要明确写“深度价值模式（受限）”
